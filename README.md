## Getting Started

### Prerequisites

- Flutter SDK (3.19 or higher)
- Dart SDK (3.3 or higher)

### Installation

1. Clone the repository
2. Install dependencies:
```bash
flutter pub get
```

3. Run the app:
```bash
flutter run
```

### Running Tests

```bash
flutter test
```
